# Future Process Analysis (To-Be Process)

## Proposed Passenger Journey

The proposed airport passenger journey focuses on reducing delays, improving passenger flow, and enabling proactive operational management through analytics.

1. Passenger Arrival
2. Self Check-in / Counter Check-in
3. Smart Security Screening
4. Immigration Processing
5. Boarding Gate Management
6. Aircraft Boarding
7. Destination Arrival
8. Optimized Baggage Claim

## Proposed Improvements

### Check-in
- Introduce self-service kiosks.
- Dynamic counter allocation during peak hours.

### Security Screening
- Predict queue lengths using analytics.
- Allocate additional staff during congestion.

### Immigration
- Improve staffing allocation using passenger forecasts.
- Monitor processing times in real time.

### Boarding
- Introduce staggered boarding groups.
- Monitor boarding efficiency using KPIs.

### Baggage Claim
- Predict baggage arrival times.
- Optimize carousel assignment.

## Expected Outcomes

- Reduced waiting times.
- Improved passenger satisfaction.
- Better resource utilization.
- Faster operational decision-making.
- Improved airport efficiency.
