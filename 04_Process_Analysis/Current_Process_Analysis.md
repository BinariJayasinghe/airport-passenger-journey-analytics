# Current Process Analysis (As-Is Process)

## Current Passenger Journey

The current airport passenger journey consists of the following stages:

1. Passenger Arrival
2. Check-in
3. Security Screening
4. Immigration Clearance
5. Boarding Gate Processing
6. Aircraft Boarding
7. Destination Arrival
8. Baggage Claim

## Identified Pain Points

### Check-in
- Long queues during peak hours.
- Insufficient counters during busy periods.

### Security Screening
- Delays caused by manual inspections.
- Limited screening lanes.

### Immigration
- Long processing times for international travelers.
- Staffing shortages during peak periods.

### Boarding
- Congestion near boarding gates.
- Inefficient boarding group management.

### Baggage Claim
- Delays in baggage delivery.
- Carousel congestion.
