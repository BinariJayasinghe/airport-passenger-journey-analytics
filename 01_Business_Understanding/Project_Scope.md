# Project Scope

## Project Overview

The Airport Passenger Journey Analytics project aims to analyze passenger movement across key airport touchpoints and provide actionable insights to improve operational efficiency and passenger experience.

---

## In Scope

The project will include analysis of:

- Passenger check-in process
- Security screening process
- Immigration processing
- Boarding operations
- Baggage claim operations
- Passenger satisfaction metrics
- Waiting times and queue analysis
- Peak hour passenger traffic analysis
- Operational KPI monitoring
- Power BI dashboard development

---

## Out of Scope

The following areas are excluded from this project:

- Aircraft maintenance operations
- Airline ticket pricing analysis
- Flight scheduling optimization
- Airport retail operations
- Fuel management systems
- Air traffic control operations
- Aircraft turnaround management

---

## Project Deliverables

The project will deliver:

- Business Case
- Stakeholder Analysis
- Requirements Documentation
- Process Analysis
- Data Dictionary
- SQL Analysis
- Power BI Dashboard
- Business Insights
- Recommendations
- Final Report

---

## Project Boundaries

The project focuses only on the passenger journey from:

Check-in → Security → Immigration → Boarding → Baggage Claim

Other airport operations fall outside the scope of this initiative.
